# GPU 측정기 기술 설계서

## 상태

proposed

## 목적

현재 컴퓨터에서 실행 가능한 GPU 측정 작업을 정의하고, 측정 결과를 다른 GPU의 정적 벤치마크 데이터와 비교할 수 있는 형태로 저장한다.

## 책임

- 실행 중인 GPU와 환경 정보를 확인한다.
- 정해진 측정 작업을 실행한다.
- 측정 시작·종료 시각과 결과를 기록한다.
- 측정 결과를 비교 엔진이 사용할 수 있는 구조로 반환한다.

## 입력

- 측정할 작업 또는 벤치마크 이름
- 실행 시간 또는 반복 횟수
- 현재 GPU 정보
- 측정 환경 설정

## 출력

- GPU 사용률
- 메모리 사용량
- 소비 전력 및 온도
- 측정 작업의 실행 시간과 처리량
- 측정 시각과 실행 환경

## 처리 흐름

1. 측정 환경과 GPU 정보를 확인한다.
2. 측정 시작 상태를 기록한다.
3. 지정된 작업을 실행하며 성능 지표를 수집한다.
4. 측정 종료 상태를 기록한다.
5. 원시 결과와 요약 결과를 저장한다.

## 구현 범위

- 1차 대상은 NVIDIA GPU와 `nvidia-smi` 기반 정보 수집으로 한다.
- 측정 결과는 정적 벤치마크 데이터와 구분한다.
- 동일한 작업을 반복 실행할 수 있도록 측정 조건을 함께 저장한다.

## 완료 조건

- [ ] 현재 GPU와 측정 환경을 식별할 수 있다.
- [ ] 하나의 측정 작업을 실행하고 결과를 저장할 수 있다.
- [ ] 측정 결과에 실행 조건과 시각이 포함된다.
- [ ] 저장된 결과를 비교 작업에서 읽을 수 있다.
