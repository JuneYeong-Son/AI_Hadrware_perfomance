## 작업 목적

- GPU Measurer MVP를 실행 가능한 상태로 구현한다.
- 제조사, 웹, AI 에이전트로 확장할 수 있는 코드 경계를 만든다.

## 수행 내용

- GPU 벤치마크 CSV와 문서 디렉터리를 정리했다.
- GPU-Z 흐름을 참고한 네 탭 데스크톱 UI를 구현했다.
- NVIDIA GPU 정적 정보와 실시간 센서 수집을 구현했다.
- PassMark 및 Compute API CSV의 정확한 모델 매칭을 구현했다.
- 센서 집계, CSV 기록, 시간 측정 로그를 구현했다.
- Device details 세로 스크롤바와 UI 종료 시 스레드 안전성을 보완했다.
- `GpuCollector` 공통 계약과 `CollectorRegistry`를 구현했다.
- UI와 향후 웹·에이전트가 공유할 `GpuMeasurementService`를 구현했다.
- JSON 직렬화 계층과 `health/list/inspect/snapshot/report` CLI를 구현했다.
- `--no-save`로 파일 생성 없는 에이전트 측정을 지원했다.
- 공식 NVML, AMD SMI, Intel Sysman, MCP 도구 규격을 조사했다.
- `logs`를 `backlogs`로 변경하고 요청 기록을 `worklog`로 이동했다.
- 웹·AI 에이전트 개발 실행 백로그를 작성했다.
